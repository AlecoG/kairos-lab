# Fotos de productos

Coloca aquí las fotos reales de los productos en formato `.jpg`, `.png` o `.webp`.

Nombres cargados actualmente:
- `cera-texturizante.png`
- `shampoo-anticaspa.png`
- `aceite-para-barba.png`

Recomendación:
- Tamaño sugerido: 1200x800 px (relación 3:2).
- Compresión web para cargar rápido.

Si falta una foto, la web usa automáticamente una imagen de respaldo por categoría.
